═══════════════════════════════════════════════════════════════════════════════
                         RÉSUMÉ DES RELATIONS
═══════════════════════════════════════════════════════════════════════════════

1. DEPARTMENTS ──< PROFESSORS    (Un-à-Plusieurs)
   - Un département a plusieurs professeurs
   - Chaque professeur appartient à un département

2. DEPARTMENTS ──< STUDENTS      (Un-à-Plusieurs)
   - Un département a plusieurs étudiants
   - Chaque étudiant appartient à un département

3. DEPARTMENTS ──< COURSES       (Un-à-Plusieurs)
   - Un département offre plusieurs cours
   - Chaque cours appartient à un département

4. PROFESSORS ──< COURSES        (Un-à-Plusieurs)
   - Un professeur enseigne plusieurs cours
   - Chaque cours est enseigné par un professeur

5. STUDENTS ──< ENROLLMENTS >──  COURSES (Plusieurs-à-Plusieurs)
   - Un étudiant peut s'inscrire à plusieurs cours
   - Un cours peut avoir plusieurs étudiants
   - ENROLLMENTS est la table de jonction

6. STUDENTS ──< GRADES >──  COURSES (Plusieurs-à-Plusieurs)
   - Un étudiant peut avoir des notes dans plusieurs cours
   - Un cours peut avoir des notes pour plusieurs étudiants
   - GRADES stocke les enregistrements d'évaluation

═══════════════════════════════════════════════════════════════════════════════
                    DÉFINITIONS DES CHAMPS CLÉS
═══════════════════════════════════════════════════════════════════════════════

STUDENTS.matricule:
  - Numéro d'inscription unique de l'étudiant
  - Format: 232332029109 (12 chiffres)


STUDENTS.level:
  - L1 = Première année (Licence 1)
  - L2 = Deuxième année (Licence 2)
  - L3 = Troisième année (Licence 3)
  - M1 = Première année Master
  - M2 = Deuxième année Master

ENROLLMENTS.status:
  - Active = Actuellement inscrit
  - Passed = Terminé avec succès
  - Failed = N'a pas réussi
  - Withdrawn = A abandonné le cours

GRADES.evaluation_type:
  - Contrôle = Examen de mi-semestre (40%)
  - Examen = Examen de fin de semestre (60%)
  - TP = Travaux Pratiques
  - TD = Travaux Dirigés

GRADES.grade:
  - Note sur 20 (système algérien)
  - Note minimale de passage: 10/20
  - Excellent: 16-20
  - Bien: 14-16
  - Assez bien: 12-14
  - Passable: 10-12

GRADES.coefficient:
  - Poids de l'évaluation
  - Typiquement 0.4 pour contrôle, 0.6 pour examen
  - Utilisé pour calculer la moyenne pondérée


═══════════════════════════════════════════════════════════════════════════════
                       CONTRAINTES DE TABLES
═══════════════════════════════════════════════════════════════════════════════

CLÉS PRIMAIRES (PK):
  - Toutes les tables ont une clé primaire entière auto-incrémentée

CLÉS ÉTRANGÈRES (FK):
  - professors.department_id → departments.department_id
  - students.department_id → departments.department_id
  - courses.department_id → departments.department_id
  - courses.professor_id → professors.professor_id
  - enrollments.student_id → students.student_id
  - enrollments.course_id → courses.course_id
  - grades.student_id → students.student_id
  - grades.course_id → courses.course_id

CONTRAINTES UNIQUES:
  - professors.email (doit être unique)
  - students.email (doit être unique)
  - students.matricule (doit être unique)
  - courses.course_code (doit être unique)
  - enrollments(student_id, course_id) - unique composite

INDEX:
  - 8 index créés sur les clés étrangères pour optimisation


═══════════════════════════════════════════════════════════════════════════════
                    RÈGLES MÉTIER APPLIQUÉES
═══════════════════════════════════════════════════════════════════════════════

1. Chaque professeur doit appartenir à un département
2. Chaque étudiant doit appartenir à un département
3. Chaque cours doit être enseigné par un professeur et appartenir à un département
4. Un étudiant ne peut s'inscrire qu'une seule fois à un cours (contrainte unique)
5. Les notes sont enregistrées par étudiant par cours par type d'évaluation
6. Les adresses e-mail doivent être uniques pour tous les professeurs et étudiants
7. Les codes de cours doivent être uniques
8. Les matricules d'étudiants doivent être uniques
9. Note minimale: 0, Note maximale: 20
10. Coefficient entre 0 et 1 pour les pondérations
